API

Creation

/api/new

You can programmatically create pastes using this route.
Accepts multipart/form-data, with the field contents as the paste content. The max upload size is 20mb.
API returns {"id": "<paste ID>", "revoke": "<revoke token>"} if the request suceeded. If the request failed, it returns a human-readable error.

Deletion

/api/delete/<id>/<token>

The revoke token gotten from the creation route can be used to delete pastes. It returns a human-readable string and a response code.
